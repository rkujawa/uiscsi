4. Create `uiscsi-ls/format.go` (package main):
- Define result types used by both formatters and the probe layer:
```go
type PortalResult struct {
    Portal  string         `json:"address"`
    Targets []TargetResult `json:"targets"`
    Err     error          `json:"-"`
}

type TargetResult struct {
    IQN  string      `json:"iqn"`
    LUNs []LUNResult `json:"luns"`
    Err  error       `json:"-"`
}

type LUNResult struct {
    LUN          uint64 `json:"lun"`
    DeviceType   uint8  `json:"device_type_code"`
    DeviceTypeS  string `json:"device_type"`
    Vendor       string `json:"vendor"`
    Product      string `json:"product"`
    Revision     string `json:"revision"`
    CapacityBytes uint64 `json:"capacity_bytes,omitempty"`
    BlockSize    uint32 `json:"block_size,omitempty"`
    LogicalBlocks uint64 `json:"capacity_blocks,omitempty"`
    CapacityStr  string `json:"-"`
}
```

- Define `func formatCapacity(blocks uint64, blockSize uint32) string`:
  - Compute `bytes := blocks * uint64(blockSize)`
  - >= 1e12 -> "%.2fTB", >= 1e9 -> "%.2fGB", >= 1e6 -> "%.2fMB", else "%dB"
  - Use SI (decimal) units per lsscsi convention

- Define `func outputColumnar(w io.Writer, results []PortalResult)`:
  - Create `tabwriter.NewWriter(w, 0, 0, 2, ' ', 0)`
  - For each portal -> target -> LUN, write one line: `"%s\t%s\tLUN %d\t%s\t%-8s\t%-16s\t%-4s\t%s\n"` with fields: IQN, portal address, LUN number, device type name, vendor (trimmed), product (trimmed), revision (trimmed), capacity string
  - For portals with Err != nil, print to stderr: `fmt.Fprintf(os.Stderr, "error: %s: %v\n", pr.Portal, pr.Err)`
  - For targets with Err != nil, print to stderr: `fmt.Fprintf(os.Stderr, "error: %s on %s: %v\n", tr.IQN, pr.Portal, tr.Err)`
  - Call `tw.Flush()` at the end
  - Per D-01: output one line per LUN showing target IQN, portal, LUN number, device type, vendor, product, revision, capacity

- Define `func outputJSON(w io.Writer, results []PortalResult)`:
  - Use `json.NewEncoder(w)` with `SetIndent("", "  ")`
  - Wrap results in `struct{ Portals []PortalResult }` with json tag `"portals"`
  - Per D-02: machine-parseable JSON output

- Add necessary imports: `encoding/json`, `fmt`, `io`, `os`, `strings`, `text/tabwriter`
