  <action>
    1. **Add SendTargets method to discovery.go** (per DISC-01):
       - `(s *Session) SendTargets(ctx context.Context) ([]DiscoveryTarget, error)`:
         1. Acquire CmdSN from s.window.acquire(ctx) (SendTargets is a non-immediate text command)
         2. Register ITT via s.router.Register()
         3. Build TextReq PDU: Final=true, ITT=itt, TTT=0xFFFFFFFF (initiator-initiated), CmdSN=acquired, ExpStatSN=s.getExpStatSN(). Data segment = login.EncodeTextKV([]login.KeyValue{{Key: "SendTargets", Value: "All"}})
         4. Encode via pdu.EncodePDU, build RawPDU, send to writeCh
         5. Wait on Router channel with timeout (30s or ctx deadline):
            - Receive TextResp, decode from raw
            - Accumulate data segment
            - Per Pitfall 6 from RESEARCH.md: if TextResp Continue bit (C=1), must send another TextReq with TTT from response to get next chunk. Loop until C=0 (Final=true). For continuation: build TextReq with Final=true, ITT from new Register, TTT=resp.TargetTransferTag, CmdSN from acquire, ExpStatSN.
            - When Final response received (C=0): parse accumulated data via parseSendTargetsResponse
         6. Update sequence numbers from response
         7. Return parsed []DiscoveryTarget, nil

    2. **Add Discover convenience function** (per D-06):
       - `Discover(ctx context.Context, addr string, opts ...login.LoginOption) ([]DiscoveryTarget, error)`:
         Standalone function (not a method) that performs full discovery in one call:
         1. `transport.Dial(ctx, addr)` -- connect
         2. `login.Login(ctx, conn, append(opts, login.WithSessionType("Discovery"))...)` -- login as discovery session. Prepend WithSessionType so user opts can override if needed.
         3. `NewSession(conn, *params)` -- create session
         4. `session.SendTargets(ctx)` -- get targets
         5. `session.Logout(ctx)` -- clean logout (or just Close if Logout fails)
         6. Return targets
         On any error: clean up (close connection) and return error.

    3. **Add integration tests to discovery_test.go**:
       Using mock target over net.Pipe():
       - `TestSendTargetsSingleResponse`: mock target returns TextResp with Final=true containing 2 targets, verify parsed correctly
       - `TestSendTargetsContinuation`: mock target returns TextResp with C=1 (partial data), then Final TextResp with rest. Verify all targets returned. This tests Pitfall 6 (C-bit continuation).
       - `TestSendTargetsEmpty`: mock target returns TextResp with empty data segment, verify empty slice returned
       - `TestDiscoverIntegration`: full Discover() test with mock target that handles discovery login + SendTargets + logout. Use net.Listener on loopback for realistic TCP. This is the most comprehensive test -- it verifies the full Dial->Login->SendTargets->Logout flow.
