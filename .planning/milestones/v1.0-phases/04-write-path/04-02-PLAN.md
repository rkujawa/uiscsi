    Create internal/session/dataout.go with three methods on the task struct:

    **1. sendDataOutBurst** -- core function that sends a burst of Data-Out PDUs:
    ```go
    // sendDataOutBurst reads up to burstLen bytes from t.reader and sends them
    // as Data-Out PDUs via writeCh. Each PDU's data segment is bounded by
    // maxRecvDSL. Returns the number of bytes sent. DataSN starts at 0 for
    // each burst per RFC 7143.
    func (t *task) sendDataOutBurst(writeCh chan<- *transport.RawPDU,
        ttt uint32, offset uint32, burstLen uint32, maxRecvDSL uint32,
        expStatSN func() uint32) (uint32, error)
    ```
    Implementation:
    - dataSN starts at 0 (per-burst, Pitfall 2)
    - Loop: while sent < burstLen
      - chunkSize = min(maxRecvDSL, burstLen-sent)
      - buf = transport.GetBuffer(int(chunkSize))
      - n, err = io.ReadFull(t.reader, buf[:chunkSize])
      - Handle io.ErrUnexpectedEOF (last chunk shorter than requested): use buf[:n], this is the final chunk
      - Handle other errors: transport.PutBuffer(buf), return error (per D-09)
      - If n == 0: break (no more data)
      - Build pdu.DataOut with: ITT=t.itt, TTT=ttt, DataSN=dataSN, BufferOffset=offset+sent, ExpStatSN=expStatSN(), Final=(sent+uint32(n) >= burstLen or last data from reader), DataSegmentLen=uint32(n), Data=buf[:n]
      - Marshal to BHS, create transport.RawPDU{BHS: bhs, DataSegment: buf[:n]}
      - Send via writeCh
      - dataSN++, sent += uint32(n)
      - If err == io.ErrUnexpectedEOF: break (reader exhausted)
    - Return sent, nil

    **2. handleR2T** -- processes an R2T PDU (called from taskLoop):
    ```go
    // handleR2T processes an R2T PDU by reading data from the task's io.Reader
    // and sending Data-Out PDUs. Enforces MaxBurstLength per WRITE-05.
    func (t *task) handleR2T(r2t *pdu.R2T, writeCh chan<- *transport.RawPDU,
        expStatSN func() uint32, params login.NegotiatedParams) error
    ```
    Implementation:
    - desired = r2t.DesiredDataTransferLength
    - Cap desired at params.MaxBurstLength (WRITE-05): `if desired > params.MaxBurstLength { desired = params.MaxBurstLength }`
    - Call t.sendDataOutBurst(writeCh, r2t.TargetTransferTag, r2t.BufferOffset, desired, params.MaxRecvDataSegmentLength, expStatSN)
    - Return error from sendDataOutBurst

    **3. sendUnsolicitedDataOut** -- sends unsolicited Data-Out after command PDU:
    ```go
    // sendUnsolicitedDataOut sends unsolicited Data-Out PDUs (TTT=0xFFFFFFFF)
    // when InitialR2T=No. The burst is bounded by FirstBurstLength minus any
    // immediate data already sent (tracked in t.bytesSent).
    func (t *task) sendUnsolicitedDataOut(writeCh chan<- *transport.RawPDU,
        expStatSN func() uint32, params login.NegotiatedParams) error
    ```
    Implementation:
    - remaining = params.FirstBurstLength - t.bytesSent (Pitfall 1: immediate data counts toward FirstBurstLength)
    - If remaining <= 0: return nil (immediate data consumed the entire first burst)
    - ttt = 0xFFFFFFFF (unsolicited marker)
    - offset = t.bytesSent (continue from where immediate data left off)
    - sent, err = t.sendDataOutBurst(writeCh, ttt, offset, remaining, params.MaxRecvDataSegmentLength, expStatSN)
    - t.bytesSent += sent
    - Return err

    Then modify session.go:

    **4. Extend taskLoop** to handle R2T PDUs for write tasks:
    In the switch on decoded PDU type, add a case for *pdu.R2T:
    ```go
    case *pdu.R2T:
        s.window.update(p.ExpCmdSN, p.MaxCmdSN)
        s.updateStatSN(p.StatSN)
        if err := tk.handleR2T(p, s.writeCh, s.getExpStatSN, s.params); err != nil {
            tk.cancel(fmt.Errorf("session: write data: %w", err))
            s.cleanupTask(tk.itt)
            return
        }
    ```
    Note: `s.getExpStatSN` is passed as a `func() uint32` so each Data-Out PDU gets the latest ExpStatSN.

    **5. Send unsolicited Data-Out after command in Submit** (or right after starting taskLoop):
    In Submit, after sending the SCSICommand via writeCh and before `go s.taskLoop(...)`, add unsolicited Data-Out dispatch:
    ```go
    // Send unsolicited Data-Out when InitialR2T=No (per D-04, WRITE-04)
    if isWrite && !s.params.InitialR2T {
        if err := tk.sendUnsolicitedDataOut(s.writeCh, s.getExpStatSN, s.params); err != nil {
            tk.cancel(fmt.Errorf("session: unsolicited data: %w", err))
            s.cleanupTask(tk.itt)
            return nil, fmt.Errorf("session: unsolicited data: %w", err)
        }
    }
    ```
    This runs synchronously in Submit (before starting the task goroutine), so the io.Reader is still in the Submit goroutine. After unsolicited data is sent, the io.Reader transitions to exclusive task goroutine ownership for R2T handling. Per Pitfall 6, no concurrent reads.
