**Structured slog lifecycle logging for session/login stages, DigestError connection-fatal behavior per D-03, and ITT/CmdSN-enriched error messages**

## Accomplishments
- Session lifecycle logging (open, close, reconnect start/complete/fail, command window changes) at Info/Warn level via injected logger

- Login stage transition logging at Info level (security negotiation, operational negotiation, full feature phase)
- Error messages enriched with ITT, CmdSN, offset, and DataSN context for protocol debugging

## Decisions Made
- DigestError is unconditionally connection-fatal (no reconnect) per RFC 7143 Section 7.3 -- digest corruption means the TCP stream is untrustworthy
- Command window changes only logged when MaxCmdSN actually changes to avoid noisy per-PDU logging
- Login logger injected via WithLoginLogger option rather than requiring it -- defaults to slog.Default() for backward compatibility
- captureHandler pattern for test assertions on slog output (shared between session and login test files)
