type Target struct {

}

**Skip helpers** (Claude's discretion):
- `RequireRoot(t *testing.T)` -- checks `os.Getuid() != 0`, calls `t.Skip` with clear message
- `RequireModules(t *testing.T)` -- reads `/proc/modules`, checks for `target_core_mod`, `iscsi_target_mod`, `target_core_file`, skips with message naming missing module
- `RequireConfigfs(t *testing.T)` -- checks `/sys/kernel/config/target/iscsi` exists

**Setup function** (per D-01, D-02):
`func Setup(t *testing.T, cfg Config) (*Target, func())` -- returns Target and explicit cleanup func (NOT t.Cleanup per D-02).

Setup sequence (per RESEARCH.md verified configfs order):
1. Call RequireRoot, RequireModules, RequireConfigfs (skip if any fail)
2. Generate unique backstore names using `crypto/rand` hex suffix
3. Allocate ephemeral port: `net.Listen("tcp", "127.0.0.1:0")`, extract port, close listener (per D-05)
4. Pre-create backing files in `/dev/shm/` with `os.Create` + `file.Truncate(size)` (per D-04)
5. Create fileio backstore: mkdir under `coreBase/fileio_0/<name>`, write `fd_dev_name=<path>,fd_dev_size=<bytes>` to `control`, write `1` to `enable`
6. Create iSCSI target IQN: mkdir `iscsiBase/<iqn>`
7. Create TPG: mkdir `iscsiBase/<iqn>/tpgt_1`
8. Create network portal: mkdir `iscsiBase/<iqn>/tpgt_1/np/127.0.0.1:<port>` (per D-05)
9. For each LUN: mkdir `lun_N`, symlink to backstore
10. Create ACL: mkdir `acls/<initiatorIQN>`
11. For each LUN: mkdir mapped LUN in ACL, symlink to TPG LUN
12. If CHAP configured: set `attrib/authentication` to `1`, write `auth/userid`, `auth/password` (per D-03). For non-CHAP: set `attrib/generate_node_acls` to `1`, `attrib/demo_mode_write_protect` to `0`.
13. If mutual CHAP: also write `auth/userid_mutual`, `auth/password_mutual`, `auth/authenticate_target` to `1`
14. Enable TPG: write `1` to `enable`

**Cleanup func** (per D-02, D-09):
The returned cleanup func tears down in STRICT reverse order (per RESEARCH.md Pitfall 1):
1. Disable TPG (echo 0 > enable)
2. Remove ACL mapped LUN symlinks, then mapped LUN dirs
3. Remove ACL dirs
4. Remove LUN backstore symlinks, then LUN dirs
5. Remove network portal
6. Remove TPG, then IQN
7. Remove backstore dir
8. Remove /dev/shm backing files

Use `os.Remove` for each item (NOT `os.RemoveAll` -- configfs doesn't support recursive removal per RESEARCH.md anti-pattern). Log errors but don't fail (best-effort cleanup). The cleanup func also scans for any targets matching IQNPrefix (per D-09) and tears down orphans.

**IMPORTANT:** All configfs writes use `[]byte("value")` without trailing newline (per RESEARCH.md anti-pattern). Use `os.WriteFile` with perm `0o644`.

Create `test/lio/sweep.go` with `//go:build e2e` tag (per D-07):
- `func SweepOrphans()` -- reads `iscsiBase` directory, finds entries with `IQNPrefix`, tears down each using same reverse-order logic. Returns error but callers should log, not fail (orphans from crashed runs per D-09).
- Helper `teardownTarget(iqn string)` that does best-effort teardown of a single target IQN regardless of state (handles partial teardown where some components may not exist).

All file operations use `os.MkdirAll`, `os.WriteFile`, `os.Symlink`, `os.Remove` per D-03 -- no targetcli, no shell commands.
